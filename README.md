# Career360-Landing
A modern, responsive landing page built as part of a frontend assignment to demonstrate UI/UX design, responsive layouts, and real-world web development practices.

🔗 [View Live Website](https://career360landing.netlify.app/) 

Overview

This project is a fully responsive landing page for CareerChoice360, designed to simulate a real-world education platform.

The goal was to create a clean, conversion-focused UI with interactive elements and structured code.

### Features: 
Hero section with strong call-to-action </br>
Courses / Specializations with interactive cards </br>
Dynamic Apply Modal with validation </br>
FAQ section with accordion behavior </br>
Fully responsive (mobile + desktop) </br>
Smooth scrolling navigation </br>
Modern UI with gradients, shadows, and hover effects </br>

### Tech Stack: 
HTML5 </br>
CSS3 </br>
JavaScript (Vanilla JS) </br>
Font Awesome (icons) </br>
Google Fonts (Inter) </br>
